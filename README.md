#  Explainable AI-Based Real-Time Fraud Detection System

>  An end-to-end, real-time, explainable AI system for detecting and monitoring financial fraud with automated retraining and drift detection.

---

##  Overview

Financial fraud is a major challenge in the FinTech industry, leading to significant financial losses every year.  
This project presents a **complete MLOps pipeline** that detects fraudulent transactions in real time while ensuring **transparency, monitoring, and continuous improvement**.

---

##  Key Features

-  Real-time fraud prediction via FastAPI  
- Interactive Streamlit dashboard  
- Automated retraining pipeline  
-  Data drift detection using Evidently AI  
-  Transaction logging with SQLite  
-  Model tracking and versioning with MLflow  
-  Explainable AI (SHAP) *(in progress)*
